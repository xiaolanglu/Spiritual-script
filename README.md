# 🔮 灵界时辰天道罗盘 (Spiritual Script)
> **“修仙不计年，转瞬已千秋。然感知灵界大势，当自洞察时辰始。”**
> 
**灵界时辰天道罗盘** 是一款专为网页挂机修仙游戏打造的 **拟物化、重工业级视觉交互** Tampermonkey 用户脚本。本法宝摒弃了传统的粗糙数字外显，通过引动 **天道四象流体大阵**，将游戏内的古法十二时辰（子丑寅卯...）演化为具有呼吸感、液态翻涌、物理光影联动以及高级空间透视的掌上微型罗盘。
## 🌌 法宝异象 (核心特性)
 * **🔮 拟物级·双层流体玉莹大成** 摒弃粗糙的静态贴图，核心采用双层非线性 radial-gradient 撕裂涡流算法（主流体顺时针揉杂，次流体逆时针对冲），达成如同活体灵液、汞水翻涌的极高视觉精细度。
 * **🎨 四象时辰·天道界域随心变幻** 根据游戏实时变化的十二时辰，自动无缝切换四种极具仙侠美学的状态机：
   * **【白昼 · 日耀玉白】**：温润如玉，日光和煦。
   * **【破晓 · 天青苍穹】**：天青有雨，灵光乍现。
   * **【黄昏 · 夕照熔金】**：晚霞漫金，烈火熔岩。
   * **【重铸 · 玄青夜色】**：摒弃凡俗之紫，采用**寒潭玄青与幽冥墨蓝**色轨，冷冽深邃，悬停时更会爆发出**月华淡银镶边**与冷翠高光。
 * **👁️ 空间透视·真言破水三维控制** 实装第四方向美学微调，古篆真言拥有了**纵深空间层次感**：
   * **静止时**：自带微距发散软阴影，文字悬浮于真玉镜面之上 1 毫米。
   * **悬停时**：神识下压，文字微缩并产生下沉硬阴影，产生**破开水面、沉入灵液**的视觉错觉。
   * **点击时**：真言深度嵌入流体核心，阴影极度收敛，达成**融合归一**的物理刚感。
 * **🖐️ 严密物理拖拽与智能折叠** 支持全平台（PC 鼠标 / 移动端触控）流畅拖拽。松开后触发智能边缘吸附。当罗盘静止 5 秒且处于视口边缘时，会自动收纳并**纳芥半隐（折叠 68%）**，同时延伸出隐形神识感应盾牌，防止打扰道友修仙挂机。
 * **🚫 摒弃多余外显，回归纯净** 彻底抹除了浏览器原生的 title 悬停黑边提示框，防止惊扰天道，保持画面极致纯净。
## 🛠️ 淬炼法理 (技术架构)
 * **字体引动**：动态灌注 Google 云端古法篆书字体大阵（Noto Serif TC / ShuowenZuan），确保真言苍劲有力。
 * **性能关隘**：核心动画全量由 GPU（CSS 3D Transform 与 Skew 混合算法）引动，在缩放、翻涌、震荡时达成高帧率、低功耗。
 * **感知机制**：采用高级 MutationObserver 嵌入游戏底层时辰 DOM（#headerGameTime），防抖毫秒级响应，绝不漏掉一丝天道异动。
 * **位移固化**：借助 localStorage 阵法，法宝位置一经道友摆放，便会牢牢锁死在对应界域，刷新不迷失。
